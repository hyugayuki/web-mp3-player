# Web MP3 Player

ブラウザで動作するシングルファイルの音楽プレイヤーです。ビルド不要、依存パッケージなしで利用できます。

## 機能

- **再生コントロール** — 再生/一時停止、音量調整、再生速度変更（0.5x〜2x）
- **スキップ** — 前後 5秒 / 10秒 / 30秒のスキップボタン
- **A-Bリピート** — 区間を指定してループ再生（プログレスバー上に範囲を表示）
- **再生履歴** — 最近開いたファイルを最大20件保存（IndexedDB / localStorage）
- **オーディオビジュアライゼーション** — Web Audio APIによる周波数連動のグロウエフェクト
- **キーボードショートカット**
  - `Space` — 再生/一時停止
  - `←` / `→` — 5秒スキップ
  - `A` / `B` — A-Bリピートのポイント設定
  - `Esc` — A-Bリピート解除 / モーダルを閉じる

## 対応フォーマット

MP3, WAV, OGG, FLAC, M4A

## 使い方

`index.html` をブラウザで開くだけで利用できます。

```sh
open index.html
```

## 動作環境

Chrome, Firefox, Safari, Edge など Web Audio API に対応したモダンブラウザ。
File System API 対応ブラウザ（Chrome, Edge）ではネイティブのファイルピッカーが利用できます。
